Users - CRUD

Create User 

Request

{
  name: "vinicius",
  email: "vinicius@mail.com", 
  password: "123456",
  isAdm: True, default=False
  address: {
    country: "Brasil",
    state: "RJ"
    city: "Araruama",
    road: "Rua do morador"
  }
}

Return 

{
  id: 1
  name: "vinicius",
  email: "vinicius@mail.com",
  isAdm: True
  address: {
    id: 1
    country: "Brasil",
    state: "RJ"
    city: "Araruama",
    road: "Rua do morador"
  }
}

Edit User - Só o usuario dono da conta e o Adm podem editar (TOKEN)

  address: {
    id: 1

}

Products

Create Products - Precisa de token

Request

{
  name: "PC gamer",
  category: "eletronicos",
  descrition: "PC Gamer muito bom",
  stock: 2
}

Return

{
  id: "1"
  name: "PC gamer",
  category: "eletronicos",
  descrition: "PC Gamer muito bom",
  stock: 2,
  user: {
    id: 1
    name: "vinicius",
    email: "vinicius@mail.com",
  }
}
